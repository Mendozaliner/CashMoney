# Archive — legacy prototype

`MACD_Strategy.py` + `AMZN_daily.csv` are the project's original single-stock
MACD prototype (tested only on Amazon). They are kept for history but are NOT
part of the live research system, which trades a broad-universe strategy scored
by walk-forward, deflated-Sharpe, and bootstrap significance. The figure PNGs
the old README referenced were never committed; the code prints its own charts
when run. Do not extend this prototype — see the top-level README and STATE.md.
